WAOULA CMS V3 — mise à jour sans recréer GitHub ni Supabase

1. Remplace index.html et admin.html sur GitHub.
2. Conserve config.js, assets/, .nojekyll et ton projet Supabase.
3. Dans admin.html, clique « Importer les 4 produits actuels ».
4. Les produits deviennent alors modifiables/supprimables depuis l'administration.
5. Le bouton « Supprimer tous » vide entièrement le catalogue après confirmation.
6. La galerie accepte plusieurs images en une fois.

Après chaque commit GitHub, attends 1 à 3 minutes et actualise avec F5.
